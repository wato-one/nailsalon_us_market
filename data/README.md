# Curated Reference Data

Drop trusted reference files here (PDFs, CSVs, notes). They are treated as **primary sources** alongside your Google Drive research folder.

Examples:
- Statista exports (copies)
- IBISWorld excerpts
- Client briefs
- Internal research notes

Files in this folder are tracked in git unless they are large binaries — prefer keeping large files in Google Drive and referencing them via `DRIVE_SOURCE_DIR`.
